Instructions to run the code:

1.Download the zip folder Final_Project and extract it.
2.Place the dataset "kc_house_data.csv" in the same folder.
3.Downgrade the plotly version to 3.10.0. (For maps -clustering)
  In order to do it, enter pip install plotly==3.10.0 in anaconda terminal
4.Open the python notebook (.ipynb) in Jupyter 
5.Download all necessary libraries as mentioned in the first cell of the notebook
6.Run the cells in order.
